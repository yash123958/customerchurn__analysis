# 📊 Customer Churn Analysis & Dashboard using Python & Power BI

An interactive **Customer Churn Analysis Dashboard** built using **Python** and **Power BI** to analyze customer behavior, churn patterns, payment methods, contract types, and internet services.  
The project focuses on identifying factors responsible for customer churn and generating business insights through data visualization and dashboarding.

---

# 📌 Project Objective

- Analyze customer churn behavior  
- Identify factors affecting customer retention  
- Understand the impact of contract type and payment methods  
- Analyze internet services and customer demographics  
- Build an interactive dashboard for churn analysis  

---

# 🧰 Tools & Technologies Used

- Python  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Power BI  
- Jupyter Notebook / Google Colab  
- GitHub  

---

# 📂 Dataset Overview

The dataset contains telecom customer information including:

- Customer ID

- Partner & Dependents  
- Internet Service  
- Phone Service  
- Streaming Services  
- Contract Type  
- Payment Method  
- Monthly Charges & Total Charges  
- Tenure  
- Churn Status

The following preprocessing steps were performed:

---

# 📌 Key Insights

- Month-to-month contract customers churn the most  
- Fiber Optic users show higher churn rates  
- Customers using electronic check payment method churn more frequently  
- Higher monthly charges increase churn probability  
- Long-term contracts improve customer retention  

---

# 💡 Business Recommendations

- Encourage customers to shift to long-term contracts  
- Improve customer support for Fiber Optic users  
- Provide offers for high monthly charge customers  
- Improve payment experience for electronic check users  
- Focus on customer retention strategies
